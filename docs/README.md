## 🔧 技术栈概览

- **Web 框架**: Gin - 高性能 HTTP 路由框架
- **依赖注入**: Wire - Google 官方编译时 DI 工具
- **ORM**: GORM - Go 语言功能丰富的 ORM 库
- **数据库**: MySQL - 主数据库存储
- **缓存**: Redis - 分布式缓存（验证码、会话等）
- **日志**: Zap + OpenTelemetry - 结构化日志和分布式追踪
- **配置**: Viper - 多格式配置管理
- **测试**: Testify + Gomock - 单元测试和 Mock 框架
